Pop Lang is a strongly and statically typed native language with a lightweight, Luau-shaped syntax. This book teaches the behavior implemented by compiler release **0.1.0-rc.2**.

:::caution[Release candidate]
Pop Lang is under active development. Treat this edition as a precise snapshot, not a promise about later releases.
:::

## How to use this book

Read the Language chapters in order if Pop is new to you. Use the Types and Reference chapters when you need an exact answer. Examples marked as runnable are taken from behavior exercised by the compiler or its tests.

Every source file uses the `.pop` extension.
