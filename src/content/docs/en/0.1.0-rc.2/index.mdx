# The Pop Book

This edition describes only behavior present in the **0.1.0-rc.2 compiler**. Each page is derived from compiler code and tests, and future releases get their own edition instead of silently changing old documentation.

Begin with [Welcome to Pop Lang](./start/welcome/), then follow the chapters in the sidebar. Each chapter covers one idea and stays small enough to translate independently.

- **Start** — install the toolchain and run your first program.
- **Language** — bindings, functions, control flow, data declarations, and modules.
- **Types** — every implemented primitive and compound type.
- **Reference** — keywords, operators, diagnostics, and release support.
