---

```pop
namespace Hello

function main()
    local release = "0.1.0-rc.3"
    print(`🫧 Olá da Pop {release}!`)
end
```

Pop é uma linguagem compilada e de tipagem estática, com uma sintaxe enxuta construída em torno de funções e blocos encerrados por `end`. Este livro começa com programas completos, apresenta cada ideia quando ela se torna útil e evolui até formar uma referência precisa da linguagem implementada.

Comece pela [Introdução](./start/introduction/) se Pop for novidade para você e depois [escolha seu caminho de aprendizagem](./start/learning-paths/). Quem está começando a programar pode aprender o vocabulário usado ao longo do livro, enquanto quem já programa em Luau encontra um guia dedicado ao que continua familiar e às suposições que precisam mudar. As partes posteriores abordam cada primitivo em páginas próprias, intervalos numéricos de `for`, valores condicionais, interpolação de strings, retornos múltiplos, genéricos, arrays, tabelas tipadas, registros, enums, uniões discriminadas, classes, interfaces, namespaces, atributos em tempo de compilação, execução nativa e todos os comandos disponíveis nesta edição.

Esta edição acompanha Pop 0.1.0-rc.3. Use o seletor de versão na barra de navegação ao consultar outra versão.
