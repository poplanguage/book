Pop Lang é uma linguagem nativa, forte e estaticamente tipada, com sintaxe leve inspirada em Luau. Este livro ensina o comportamento implementado pela versão **0.1.0-rc.2** do compilador.

:::caution[Versão candidata]
Pop Lang está em desenvolvimento ativo. Considere esta edição um retrato preciso, não uma promessa sobre versões futuras.
:::

## Como usar este livro

Leia os capítulos de Linguagem em ordem se Pop for novidade para você. Consulte Tipos e Referência quando precisar de uma resposta exata. Exemplos marcados como executáveis vêm de comportamentos exercitados pelo compilador ou por seus testes.

Todo arquivo-fonte usa a extensão `.pop`.
