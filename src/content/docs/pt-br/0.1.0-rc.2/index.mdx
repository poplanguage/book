# O Pop Book

Esta edição descreve somente o comportamento presente no **compilador 0.1.0-rc.2**. Cada página nasce do código e dos testes do compilador. Versões futuras recebem uma nova edição, sem alterar silenciosamente a documentação antiga.

Comece por [Boas-vindas a Pop Lang](./start/welcome/) e siga os capítulos na barra lateral. Cada capítulo cobre uma ideia e permanece pequeno o bastante para ser traduzido de forma independente.

- **Início** — instale as ferramentas e execute seu primeiro programa.
- **Linguagem** — variáveis, funções, controle de fluxo, dados e módulos.
- **Tipos** — cada tipo primitivo e composto implementado.
- **Referência** — palavras-chave, operadores, diagnósticos e suporte da versão.
